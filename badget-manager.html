Budget Manager

Introduction:
This project involves the development of a C# Desktop GUI application using WPF (Windows Presentation Foundation) to create a family budget expense manager. The application integrates with a Web API and meets several technical requirements, ensuring a robust and feature-rich budget management tool.

Technical Requirements:
The application is built using WPF for a modern and responsive user interface. It employs SQLite for data storage, ensuring a lightweight and efficient database solution. Inheritance is utilized to promote code reusability and maintainability, and data is written via an ORM (Object-Relational Mapping) for seamless database operations.

CI/CD Integration:
The project is integrated into a GitHub Actions CI/CD pipeline, allowing for continuous integration and delivery. Users can download the application as an artifact, including the .exe file and any necessary .dll files, ensuring easy deployment and updates.

Key Features:

  • User-friendly interface for managing family budgets and expenses.
  • Ability to add, edit, and delete expense records.
  • Visualization of expenses through charts and graphs for better financial insights.
  • Integration with a Web API for real-time data synchronization and updates.
  • Secure data storage using SQLite with ORM for efficient data handling.

Conclusion:
This project demonstrates the effective use of WPF, SQLite, and GitHub Actions CI/CD pipeline in building a practical and user-friendly budget management application. The integration of inheritance and ORM ensures a maintainable and scalable codebase, while the CI/CD pipeline facilitates seamless deployment and updates.
