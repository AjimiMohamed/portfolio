Student Administration Portal

Introduction:
This project involves developing a web application for Thomas More using PHP, Laravel, TailwindCSS, and MySQL. The application facilitates interactions between companies and students by allowing companies to present their recruiters' availability for chats, and enabling students to book interviews with their preferred companies.

Features for Companies:
Companies can create profiles and present their recruiters' availability for interviews. They can specify available time slots, provide information about their organization, and list the positions they are hiring for. This feature ensures that students have clear and detailed information when choosing a company to engage with.

Features for Students:
Students can browse company profiles, view available time slots, and book interviews with their preferred companies. The application allows students to manage their bookings, receive notifications for upcoming interviews, and access details about the companies and recruiters they are meeting.

Technology Stack:

  • PHP & Laravel: The backend is developed using PHP and Laravel, providing a robust and scalable framework for handling user authentication, scheduling, and data management.
  • TailwindCSS: The frontend is styled using TailwindCSS, offering a modern and responsive design that enhances user experience.
  • MySQL: The database is managed using MySQL, ensuring reliable and efficient data storage and retrieval.

Key Features:

  • User-friendly interface for both companies and students to manage profiles and bookings.
  • Secure user authentication and authorization for data protection.
  • Detailed company profiles and recruiter information to help students make informed decisions.
  • Responsive design for seamless use on various devices.

Conclusion:
This web application demonstrates the effective use of PHP, Laravel, TailwindCSS, and MySQL to create a functional and user-friendly platform for connecting companies with students at Thomas More. By streamlining the process of scheduling interviews and providing detailed information, the application enhances the overall experience for both recruiters and students.
