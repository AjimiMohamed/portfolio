Wildlife Strike in Aviation

Project Overview:
This project involves creating a comprehensive data visualization dashboard using Qlik Sense to analyze wildlife strike incidents in aviation. The goal is to tell a compelling story through data, highlighting patterns and insights related to wildlife strikes. The project runs from 07/11/2023 to 14/12/2023.

Target Audience:
The primary audience for this dashboard includes aviation safety experts, airport wildlife management teams, and regulatory authorities. The dashboard provides insights into the frequency, locations, and types of wildlife involved in strikes, helping these stakeholders make informed decisions to enhance aviation safety.

Visualized Information:
The dashboard includes multiple sheets with various visualizations, such as bar charts, line graphs, histograms, distribution plots, and box plots. It answers key questions like the most common types of wildlife strikes, seasonal patterns, and the impact of strikes on different types of aircraft.

Narrative:
The project narrates the story of wildlife strikes in aviation, exploring the patterns and trends over time. It provides actionable insights to improve safety measures and wildlife management practices at airports. The visualizations help to uncover hidden patterns and correlations in the data.

Tools and Technologies:
The dashboard is built using Qlik Sense, with extensive use of Qlik scripting for ETL (Extract, Transform, Load) processes. It includes various chart types and utilizes set expressions for complex data queries. The project also incorporates images and buttons for an enhanced user experience, adhering to the project's guidelines.
